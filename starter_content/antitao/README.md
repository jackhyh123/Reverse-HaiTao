# 反淘淘金通关系统 Starter Pack

这组文档是给 `Knowledge Vault` 的第一批种子内容。

建议上传顺序：

1. `seller-lane-foundations.md`
2. `operator-lane-foundations.md`
3. `system-map.md`

推荐用法：

- 先上传到知识仓，建立一个新的知识库，比如 `antitao-core`
- 卖家问题优先挂卖家文档，平台运营问题优先挂运营文档
- 在生成深度研究和作战手册前，先确保知识仓已经连接

这不是最终知识库，而是第一版角色化骨架，后续可以继续补：

- 站点规则
- 渠道打法
- 类目案例
- 履约 SOP
- 商家分层和平台治理规则
